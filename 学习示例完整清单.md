## 🔥 最重要的3个示例

如果时间有限，**必须掌握**这3个核心示例：

### 1️⃣ [IoCDemo.java](src/advanced/IoCDemo.java) - IoC容器
**为什么重要**：Spring框架的核心基石
- 理解依赖注入如何解耦代码
- 理解@Autowired的工作原理
- 理解Spring容器管理Bean的机制

**运行**：
```bash
cd src && java advanced.IoCDemo
```

---

### 2️⃣ [AopDemo.java](src/advanced/AopDemo.java) - AOP切面
**为什么重要**：Spring的另一大核心
- 理解横切关注点的概念
- 理解@Transactional如何工作
- 理解动态代理的原理

**运行**：
```bash
cd src && java advanced.AopDemo
```

---

### 3️⃣ [OptionalDemo.java](src/advanced/OptionalDemo.java) - Optional
**为什么重要**：现代Java避免NPE的标准做法
- Spring Data JPA大量使用Optional
- 理解如何优雅处理null

**运行**：
```bash
cd src && java advanced.OptionalDemo
```
