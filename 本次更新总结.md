## 📦 新增文件列表

### 1. NioDemo.java - NIO 文件操作 ⭐⭐⭐⭐
**文件路径**：`src/advanced/NioDemo.java`
**学习重点**：
- Path 和 Files 工具类的现代化文件操作
- FileChannel + ByteBuffer 通道缓冲区机制
- 目录遍历和文件查找（PathMatcher）
- WatchService 文件监听

**Spring 应用**：
- Spring Boot 内嵌 Netty/Tomcat 使用 NIO 处理网络 I/O
- ResourceLoader 使用 NIO 扫描类路径
- MultipartFile 文件上传底层实现
- Spring DevTools 使用 WatchService 监听文件变化

---

### 2. LoggingDemo.java - SLF4J + Logback 日志框架 ⭐⭐⭐⭐⭐
**文件路径**：`src/advanced/LoggingDemo.java`
**学习重点**：
- 日志级别（TRACE < DEBUG < INFO < WARN < ERROR）
- 参数化日志（避免字符串拼接）
- 异常堆栈记录
- MDC 诊断上下文（分布式追踪）

**Spring 应用**：
- **Spring Boot 默认日志框架**
- 统一日志规范（替代 System.out.println）
- Spring Cloud Sleuth 使用 MDC 传递 traceId
- @ControllerAdvice 全局异常日志记录

**最重要**：这是 Spring Boot 默认日志框架，必须掌握！

---

### 3. ScheduledTaskDemo.java - 定时任务 ⭐⭐⭐⭐
**文件路径**：`src/advanced/ScheduledTaskDemo.java`
**学习重点**：
- Timer vs ScheduledExecutorService 对比
- scheduleAtFixedRate vs scheduleWithFixedDelay 区别
- 定时任务的三种模式（固定速率、固定延迟、初始延迟）
- Spring @Scheduled 注解用法

**Spring 应用**：
- **@Scheduled 注解底层使用 ScheduledExecutorService**
- 定时数据同步、缓存刷新
- 定时报表生成、数据清理
- Cron 表达式配置

---

### 4. SerializationDemo.java - Java 序列化 ⭐⭐⭐⭐
**文件路径**：`src/advanced/SerializationDemo.java`
**学习重点**：
- Serializable 接口和基本序列化
- transient 关键字（字段排除）
- serialVersionUID 版本控制
- 自定义序列化（writeObject/readObject）
- 深拷贝实现

**Spring 应用**：
- **HttpSession 对象存储到 Redis 需要序列化**
- Spring Cache + Redis 缓存对象序列化
- Dubbo、Feign RPC 远程调用传递对象
- @Scope("prototype") 原型 Bean 深拷贝
